1.osi 7 계층

하위계층
physical 물리 (허브,리피터,케이블)
datalink 데이터 (NIC, L2스위치)
network 네트워크 (라우터, L3스위치)
transport 번역 (로드밸런서, 방화벽)

상위계층
session 세션
presentation 프리젠테이션
application 어플리케이션 (L7 스위치, 웹 방화벽)


★라우팅 종류 
동적라우팅 : RIP, OSPF, IGRP, EGRP, BGP (시험)

★ipconfig /all : 네트워크 주소를 확인하거나 변경하는 명령어
getmac /v : 물리적 주소를 확인
nslookup 도메인네임 : 해당 도메인 네임의 ip주소를 확인
★ping 도메인네임 : 해당 도메인네임이나 ip주소의 라우팅 경로를 확인하거나 네트워크 상태를 확인

tracert 목적지도메인 또는 ip주소  

arp -a : 물리적 주소 확인 및 수정
ARP(주소 확인 프로토콜)에서 사용하는 IP 주소에서 물리적 주소로의
변환 표를 표시하고 수정합니다.

nbtstat -a 컴퓨터이름 : 해당 컴퓨터의 이름을 가진 정보를 표시
